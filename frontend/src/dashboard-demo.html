🔗 Enhanced Dashboard - URL Status Indicators

✨ New Features Added

  • 🟠 Expired URLs: Clear orange badges and styling for URLs past their expiration date
  • 🟡 Expiring Soon: Yellow warning badges for URLs expiring within 7 days
  • 🔴 Inactive URLs: Red badges for manually deactivated URLs
  • 📅 Smart Date Display: Shows "X days ago" for expired and "X days left" for expiring
  • 🎨 Color-Coded Icons: Link icons change color based on status
  • ⚡ Real-time Status: Automatically calculates expiration based on current date

🎯 Status Logic

Active: is_active = true && (no expiration || expires_at > now)
Expiring Soon: is_active = true && expires_at <= now + 7 days
Expired: is_active = true && expires_at < now
Inactive: is_active = false
